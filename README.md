# PP FNO vs PINN

This repository contains the Programming Practical on learning the solution of a 1D backwater-flow problem and solving an associated inverse problem. The project compares Physics-Informed Neural Networks (PINNs), standard neural networks, and Fourier Neural Operators (FNOs), with the main material gathered in `fno_pinn.ipynb`.

## Clone the repository

```bash
git clone <repo-url>
cd PP-FNO-PINN
```

## Build the environment with `uv`

Install the project dependencies and create the virtual environment:

```bash
uv sync
```
